# Contributing

Thank you for helping Curious Baby grow.

## Development

```bash
npm install
npm run build
npm test
```

Use a temporary home while testing:

```bash
CURIOUS_BABY_HOME=/tmp/curious-baby-dev baby born --no-start
```

## Documentation Language

Official project documentation, CLI help text, and code comments should be written in English. Curious Baby may still support multilingual conversations with its owner.

## Pull Requests

Please include:

- A short explanation of the change.
- Tests for behavior changes.
- Documentation updates when public behavior changes.
- Notes about privacy or permission implications.
