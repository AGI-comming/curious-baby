# Dashboard

The Dashboard is a local visual control center over the same state used by the CLI.

It includes:

- Current Mind
- Chat
- Short-Term Memory
- Long-Term Memory
- Personality
- Proactive Queue
- Permissions
- Audit Log
- Growth Timeline
- Runtime Health

The v1 implementation provides the main API and an initial React UI for state, chat, memory, personality, and permission visibility.
